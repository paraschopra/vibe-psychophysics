Background

The Stroop effect, first reported by John Ridley Stroop (1935), is one of the most robust and well-replicated findings in experimental psychology. It demonstrates cognitive interference: when the name of a color word is printed in a different ink color, naming the ink color is slower and more error-prone than when the word and ink color match.

Why It Happens

Reading is a highly practiced, automatic process. When you see the word "RED" printed in blue ink, your brain automatically reads the word, creating a conflict with the task of naming the ink color. Resolving this conflict takes additional processing time.

Three Conditions

  • Congruent: Word matches ink color (e.g., RED in red ink) — fastest
  • Neutral: Non-color word (e.g., CHAIR in red ink) — intermediate
  • Incongruent: Word conflicts with ink color (e.g., RED in blue ink) — slowest

Important Note

The classic Stroop task uses vocal color naming. This keyboard version measures a related but somewhat different form of interference. Effect sizes may differ from published norms, which typically report 50–100 ms interference using vocal responses (MacLeod, 1991).

References
