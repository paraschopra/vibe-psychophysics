Background

Sternberg (1966) investigated how information is retrieved from short-term memory. Participants memorize a small set of items, then judge whether a probe item was in the set. This paradigm has become one of the most influential tools for studying memory search processes.

Key Finding

Reaction time increases linearly with set size at approximately 38 ms per item. This linear relationship suggests a serial exhaustive search — every item in the memory set is checked, even after a match is found.

Serial vs Parallel Search

If memory search were self-terminating, the slope for present trials (where the probe is in the set) would be roughly half the slope for absent trials (where every item must be checked). Sternberg found that present and absent slopes were approximately equal, providing strong evidence for exhaustive search — the entire set is scanned before a decision is made.

References

  1. Sternberg, S. (1966). High-speed scanning in human memory. Science, 153, 652–654.
  2. Sternberg, S. (1969). Memory scanning: Mental processes revealed by reaction-time experiments. American Scientist, 57, 421–457.
